IMPORTANT: the software uses positional calls so where you add something is important! Call the script from the directory it is in, we source from our main directory. 

usage: <install defult references> <path/to/user/reference> <path/to/user/annotation> <user reference name>

<custom reference>	[file]	path to reference fasta file
<custom annotation>	[file]	NCBI tabular annotation file
<name>			[Char]	Name of your reference
